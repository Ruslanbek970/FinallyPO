# HotelBookingProject (Entities + DTOs)

Это заготовка проекта в стиле Lab9 (Spring Boot + JPA + Lombok), содержит:
- 10 Entity для системы бронирования отелей/апартаментов
- Для каждой сущности Request/Response DTO

## Запуск
- Открыть как Gradle project
- Запустить `HotelBookingProjectApplication`

По умолчанию подключена H2 in-memory база (`application.properties`).
